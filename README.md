# STM32 Embedded System Project with LCD Screen Control
The purpose of this project is to develop a system that adjusts the buzzer's frequency based on ambient light intensity and changes the color of the RGB LED according to the light level. As the light intensity increases, the buzzer will beep more frequently, and the LED will turn red to alert the user.
